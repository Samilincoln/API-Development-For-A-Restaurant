The API Endpoints

Customer level API calls

  • login
  • Display only the current user
  • Generates access tokens that can be used in other API calls in this project
  • Lists all menu items.
  • Lists single menu item

Manager level API calls

  • Perform CRUD on all menu items
  • Perform CRUD on a single menu item
  • Manager group management endpoint
  • Delivery Team group management endpoint
  • Returns all orders with order items by all users

Cart Management Endpoints

  • Returns current items in the cart for the current user

Order Management Endpoint

  • Returns all order items created by this user
  • Returns all items for this order id

Delivery Team API Call

  • Returns all orders with order items assigned to the delivery crew
